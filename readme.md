# 📚 BookSky

A simple book management web application built using HTML, CSS, and JavaScript.

## Features

- Add new books
- Delete books
- Popup form for entering book details
- Dynamic DOM manipulation
- Clean and responsive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript

## What I Learned

- DOM Manipulation
- Event Listeners
- Form Handling
- Dynamic Element Creation
- Template Literals
- Popup Modals
